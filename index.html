Facts about Web Design!

  • Images are added in HTML by providing the source in which the image is actually from (the URL)
  • To add an image to an HTML, you need to add the image tag, then provide the source where the image lives by using src="". Then you can add alt="" to describe the image.
  • You can adjust the size of the image using width="" and height="". This will adjust the dimensions.
